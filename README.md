# 🎮 Rock Paper Scissors - Python

A bilingual (English/Spanish) Rock Paper Scissors game developed in Python. This project was created to practice programming fundamentals such as functions, loops, conditionals, dictionaries, error handling, and basic score management.

## ✨ Features

* 🌎 Bilingual support (English and Spanish)
* 🎯 **Best of 1** game mode
* 🏆 **Best of 3** game mode
* 📊 Score tracking system
* ⚠️ Input validation for invalid options
* 🔄 Randomized computer choices using Python's `random` module

## 🚀 Getting Started

### Prerequisites

Make sure you have Python 3 installed on your system.

Check your Python version:

```bash
python --version
```

### Installation

1. Clone this repository:

```bash
git clone https://github.com/DilanRiv/rock-paper-scissors-python.git
```

2. Navigate to the project directory:

```bash
cd rock-paper-scissors-python
```

3. Run the game:

```bash
python RPS.py
```

## 🎮 How to Play

Choose one of the following options when prompted:

* `rock` or `piedra`
* `paper` or `papel`
* `scissors` or `tijera`

### Game Modes

#### Best of 1

* Win: **+2 points**
* Lose: **-1 point**
* Draw: **0 points**

#### Best of 3

* Win the series: **+5 points**
* Lose the series: **-4 points**
